Our blockchain is powered by the Cosmos SDK, a modular framework for building blockchains. This documentation will provide an overview of our node and the Cosmos SDK, as well as detailed instructions on how to set up and use our node.

Whether you're a developer looking to build applications on our blockchain or a user looking to run a node and participate in our network, this documentation will provide the information you need to get started.
